# AmberAgent Notion-like Brand Exploration

This folder contains exploratory brand assets for the Notion-like workspace redesign line.

## Current Board

- `amberagent-notion-like-brand-exploration-2026-05-05.png`

The board explores original AmberAgent app icon, logo lockup, workspace illustration, tool/status cards, and mini icon directions. It intentionally avoids Notion's official logo, cube mark, brand lockup, Android robot imagery, and iOS-style chrome.

## Font Direction

- Brand and HTML prototype: Inter or Geist for Latin text.
- Chinese pairing: Noto Sans SC or Source Han Sans.
- Android implementation: keep the existing system sans and current chat font options for this phase; do not add a global font package yet.

## Usage Note

Treat this image as a direction board, not final production artwork. Final launcher icons should be manually selected, vectorized or recreated as adaptive icon assets, and reviewed before being wired into `res/mipmap` or `res/drawable`.
