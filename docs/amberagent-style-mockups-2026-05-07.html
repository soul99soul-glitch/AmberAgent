Color rules

Cream #F5F1EA + Ink #1A1A1A + ONE of (Chartreuse #D6FF3F | Sport-Orange #E85D2F) per surface. Reserve orange for hero numerals & critical CTAs. Reserve chartreuse for active / in-progress / primary actions.

Typography

Geometric sans (Inter / Geist). Hero numerals ~3× body, sport-orange. ALL-CAPS labels at ~9–11px, letter-spaced 0.10–0.18em, warm grey. Body medium-weight, line-height 1.5.

Layout

Card-modular. Corner radius 14–24dp. Outer 16dp, inner padding 14–20dp. No drop shadows on regular cards — only on the floating bottom-nav pill and the chartreuse send button.

Motion

Streaming = horizontal pulse-track of 7 chartreuse bars on a black mini-card. Tool running = chartreuse left-edge stripe on tan card. State changes = 220ms spring. No lottie.
