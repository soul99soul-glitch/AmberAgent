  text=Strikethrough(GFM Extension)
  text=Strikethrough text is wrapped in double tildes: ~~text~~. It is a GitHub Flavored Markdown extension and renders as an <s> or <del> HTML element.
  text=Basic Usage
  text=The old recommendation was to use AsyncTask for background work, but it has been deprecated since Android API 30. You should use Kotlin Coroutines or WorkManager instead.
  text=SharedPreferences has been superseded by DataStore for most new use cases, though SharedPreferences remains available for compatibility.

  text=Double Tilde Ambiguity
  text=A single tilde is not strikethrough: ~this is not struck~. Only double tildes trigger the extension.
  text=Two separate single tildes: text ~ with ~ single ~ tildes — all plain text.
  text=An unclosed double tilde: ~~this never closes because there is no matching delimiter on this line.
